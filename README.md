# simulateur_damage_weiss
Construction d'un simulateur dégâts pour le jeu Weiss Schwarz
Je déposerais ici les itérations au fil et à mesure de la progression (ce qui n'engage ni vous ni moi à une quelconque régularité)

La version de base contient les mécaniques suivantes :
- refresh
- level up
- trigger

Tous les paramètres sont à rentrer en manuel.
